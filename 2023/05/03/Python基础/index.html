数据容器

5种数据类型:元组(tuple),列表(list),字符串(str),字典(dict),集合(set)

列表

列表的概念：

列表是一种可变的、有序的数据结构，可以随时添加和删除其中的元素。
List（列表）是 Python 中使用最频繁的数据类型，在其他语言中通常叫做数组
专门用于存储一串信息
列表用[]定义，数据之间使用 , 分隔
列表的索引 从 0 开始

列表定义方法：

函数与方法的差别就在于定义和调用不同

常用的方法:

列表

插入元素
列表名.insert(下标值,元素)

追加元素
列表名.append(元素)

追加批量元素
列表名.extend(容器名)

删除元素
del 列表名[下标值]

删除指定元素
列表名.remove(元素)

剪切元素(不设定接收值就只是删除元素)
列表名.pop(下标值)

清空列表
列表名.clear()

统计列表特定的元素
列表名.count(元素)

查找指定元素所在的下标,找不到就报错ValueError
列表.index(元素)

统计列表有多少元素(查看列表长度)
Listlen=len(列表名)

元组

元组的概念:

一旦定义完成就不可修改。
可以存储任意类型。
定义用()，使用,隔开各个数据。
注：如果元组中只有单个元素，需要在元素后面加一个逗号
tuple_1=("hello",)

常用方法

元组

查找指定元素的下标
元组名.index("指定元素")

统计指定元素有几个
元组名.count("指定元素")

统计元素个数
tuplelen=len(元组名)

字符串

字符串的概念:

一旦定义完成就不可修改。
只能存储字符串。
允许重复字符串存在。
定义用””。

常用方法

字符串

查找指定元素的下标
字符串.index("指定元素")

替换字符串
新字符串=字符串.replace("替换前","替换后")

字符串的分割
newlist=字符串.split("分割符")
得到是一个列表

字符串的规整
字符串.strip()默认删除首尾空格
字符串.strip("参数")删除参数的值

统计字符串中某字符串出现次数
字符串.count("参数")

统计字符串的长度
strlen=len(字符串)
